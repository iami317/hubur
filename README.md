# Hubur

一个公共的安全侧 utils 库, 目前包含如下几部分：

- addr.go 地址解析
- rand.go 随机数
- string.go 字符串处理
- file.go 文件处理
- sync.go 并发同步
- time.go 时间处理
- url.go url处理
- network.go 网络连接相关
- machine.go 获取机器uuid
- netutils.go 网络处理
- parse_domain.go 域名解析
- retry.go 重试
- user_agents 随机user-agent

这个仓库会随着项目开发增多逐渐丰富，欢迎大家主动提 MR
